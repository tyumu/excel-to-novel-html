# 🚀 クイックスタート: 3ステップで完成!

## ステップ1️⃣: データを抽出（30秒）

```powershell
python simple_converter.py
```

実行すると:
- ✅ `output/ai_input.txt` が生成される（約36,000文字）
- ✅ プロンプトの例が画面に表示される

## ステップ2️⃣: AIで変換（5-10分）

### 👉 やること:

1. **プロンプトをコピー**
   - `ai_prompt.txt` を開いて全文コピー
   - または、画面に表示されたプロンプトをコピー

2. **データをコピー**
   - `output/ai_input.txt` を開いて全文コピー

3. **AIに送信**
   - GitHub Copilot Chat、ChatGPT、Claude など
   - プロンプト → データの順で貼り付けて送信

4. **結果を保存**
   - AIの出力を **`output/novel_output.txt`** に保存
   - ⚠️ ファイル名を間違えないこと!

### 💡 重要なポイント:

**AIには必ず具体例付きプロンプトを使う!**

```
❌ ダメな例: 「会話を小説にして」
   → 結果: アーミヤ「はい!」（台本形式）

✅ 良い例: ai_prompt.txt のプロンプトを使う
   → 結果: 「はい!」アーミヤが頷く。（小説形式）
```

## ステップ3️⃣: HTMLを生成（10秒）

```powershell
python simple_converter.py
```

もう一度実行すると:
- ✅ `output/generated_novel.html` が生成される
- ✅ ブラウザで開けば縦書き小説の完成!

## ⚠️ トラブルシューティング

### Q: 文字数が多すぎてAIがエラーになる

**解決策**: ファイルを分割
```powershell
# ai_input.txt を手動で2-3個のファイルに分割
# 例: ai_input_part1.txt, ai_input_part2.txt

# それぞれAIで変換

# 全部の結果を1つのファイル(novel_output.txt)に結合
```

### Q: AIが台本形式で出力してしまう

**解決策**: `ai_prompt.txt` の内容を**必ず**使う
- 具体例が書いてあるので、AIが正しく理解します

### Q: 話者情報が消えている

**解決策**: 最新版なら修正済み
```powershell
python simple_converter.py
```
を実行すると、【キャラクター名】付きで抽出されます
